After creating a basic page, we will learn how to apply styles to our elements using Cascading Style Sheets (CSS)

Cascading Style Sheets (CSS)

  1. Style Sheets save you a lot of time by allowing you to specify styles for all tags of the same type in one place.
  2. Visit this page. It demonstrates how using different css guides completely changes the look of a page.
  3. CSS Syntax is made up of a selector followed by pairs of property and values
    1. the pairs are separated by semi-colons
    2. the properties and values are separated by colons
    3. Example: 
      h2 { 
 color:brown;
 background-color: yellow;
 text-align:center;
 border: 1px solid green
}
    4. More info here
  4. You can apply styles to different types of selectors
    1. Simple selectors
      1. name
      2. id
      3. class
    2. Combinator selectors (advanced)
    3. Pseudo-classes (advanced) Used to define a state of an element (like hover)
    4. Attribute selectors (advanced)
    5. More info here
  5. CSS can be located in a document in different ways
    1. inline - as part of a tag
    2. internal - as one large style section inside the html header section
    3. external - in a separate document that is linked inside the header section of the html document
    4. More info here
  6. There are many other specific ways to use css to modify the style of your web-site. Here is a short list.
    1. Backgrounds
    2. Borders
    3. Box
    4. Text
    5. Fonts
    6. Lists
    7. Tables

    4. Shadows
