Creating Web-Sites at Camp

We will start by using Hyper Text Markup Language (HTML). The web-sites we will create will consist of the following elements/tags:

  1. Headings
  2. Text in Paragraphs
  3. Text in Tables
  4. Images
  5. Links (Text and Images)
  6. Miscellaneous

After creating a basic page, we will learn how to apply styles to our elements using Cascading Style Sheets (CSS)

If students are interested, we may learn how to create some code using javascript
